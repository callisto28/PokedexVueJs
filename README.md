## About

This repository contains a monorepo of the Pokedex app.

The Pokedex is an app which the user can view a list of Pokemons, search and view a specific Pokemon. To get the Pokemons data i was used the [Pok√©API](https://pokeapi.co/).

## Technologies

This project was developed with the following technologies:

- [VueJs](https://vuejs.org/)
- [Vue Router](https://router.vuejs.org/)
- [Vuex](https://vuex.vuejs.org/)
- [Vuetify](https://vuetifyjs.com/en/)

- [VS Code](https://code.visualstudio.com/) with [ESLint](https://eslint.org/) and [Prettier](https://prettier.io/)
